
# Umrona Test

repository untuk menyimpan code test Umrona

## Run Locally

Clone the project

```bash
  git clone https://github.com/irfanyazid-13032000/umrona.git
```

Go to the project directory

```bash
  cd umrona
```

Install dependencies

```bash
  composer install
```

```bash
  npm install
```

create a database and fill in the data
```bash
  php artisan migrate
```

```bash
  php artisan db:seed
```



Start the server

```bash
  npm run dev
```

```bash
  php artisan serve
```
